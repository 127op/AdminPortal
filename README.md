## 🧑‍💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend / Services:** Firebase Authentication, Realtime Database
- **Tools:** Firebase Hosting, OTP Verification with Firebase
- **Deployment:** Web App (Firebase Hosting)

---

## 🚀 Features

- **Admin Login** using email and password
- **OTP Verification** (2-step auth) via registered phone number
- **Real-Time User Data Retrieval** from Firebase Realtime DB
- **Search & Filter** by name, course, or registration number
- **Validation Actions**: Approve or reject students directly via UI
- **Mobile-Responsive UI**
